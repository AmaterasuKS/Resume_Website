# Resume Website

A modern, animated personal resume/portfolio website built with HTML, CSS, and vanilla JavaScript. The design focuses on premium visuals, smooth motion, and clear content hierarchy.

## Features
- Glassmorphism panels and gradient accents
- Animated hero frame with a linear gradient border
- Smooth section reveals on scroll
- Animated counters in the hero stats
- Responsive layout for desktop and mobile
- Clean contact form with focus styles

## Tech Stack
- HTML5
- CSS3 (Grid, Flexbox, gradients, animations)
- Vanilla JavaScript (IntersectionObserver, smooth scrolling)

## Project Structure
```
My Resume/
+-- index.html
+-- styles.css
+-- script.js
L-- assets/
    L-- profile.jpg
```

## Getting Started
1. Make sure `assets/profile.jpg` exists (replace with your photo).
2. Open `index.html` in a browser.

Optional: use a local server (e.g., VS Code Live Server) for the best experience.

## Customization
- Update text content directly in `index.html`.
- Adjust colors in `styles.css` under `:root`.
- Replace the hero photo in `assets/profile.jpg`.
